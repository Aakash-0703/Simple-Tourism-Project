# 🌍 Tourism Website Project

## 📌 Project Overview

This is a simple **Tourism Website** built using **HTML** and **Tailwind CSS**.
The project showcases different tourist places with a clean and responsive design using **Flexbox** for layout.

---

## 🚀 Features

* Responsive design using Tailwind CSS
* Flexbox layout for alignment and spacing
* Multiple pages (Home + destination pages)
* Clean and modern UI
* Easy navigation between pages

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* Flexbox

---

## 📂 Project Structure

```
tourism-project/
│
├── index.html          # Home page
├── places.html         # Tourist places page

```

---

## 📄 Pages Description

### 🏠 index.html

* Main landing page
* Contains hero section and featured destinations

### 📍 places.html

* Displays tourist locations
* Uses Flexbox for layout

## 🎨 Styling

* Built with Tailwind CSS utility classes
* Flex properties used:

  * `flex`
  * `justify-center`
  * `items-center`
  * `gap-*`

---

## ▶️ How to Run

1. Download or clone the project
2. Open `index.html` in your browser

---

## 🌐 Deployment (GitHub Pages)

1. Push project to GitHub
2. Go to **Settings > Pages**
3. Select branch (main)
4. Your site will be live
